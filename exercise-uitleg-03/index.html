Labo 03: CSS en Flexbox

Wat Ik Heb Leren Doen

In deze opdracht heb ik geleerd hoe ik Flexbox en CSS variabelen moet gebruiken. Het doel was om te begrijpen hoe je elementen in een layout kunt plaatsen met Flexbox en om te experimenteren met CSS Transitions. Ook heb ik geleerd hoe ik elementen visueel kan verbergen zonder dat ze helemaal verdwijnen uit de pagina, wat handig is voor toegankelijkheid. CSS nesting en logical properties kwamen ook aan bod, wat de code overzichtelijker maakt.

Mijn Ervaring

Deze opdracht was een prima manier om mijn CSS-vaardigheden te verbeteren. Flexbox en variabelen waren erg interessant en handig. Ik heb nu ook geleerd hoe ik elementen kan verbergen zonder ze helemaal uit de code te halen, wat belangrijk is als je rekening houdt met mensen die hulpmiddelen gebruiken om de website te navigeren.
