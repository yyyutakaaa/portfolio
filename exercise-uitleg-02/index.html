Labo 02 — Webpagina Ontwikkeling

Wat We Doen

In deze opdracht hebben we de basis van HTML bekeken en een introductie tot CSS gedaan. We hebben geleerd hoe we een eenvoudige webpagina kunnen opbouwen met HTML en de opmaak kunnen aanpassen met CSS.

Wat Ik Ervan Vond

Ik had al eerder met CSS gewerkt, maar het was leuk om dit opnieuw te doen als opfrissing, vooral omdat ik het een tijdje niet had gebruikt. Het was interessant om te zien hoe CSS de opmaak van een pagina kan verbeteren en hoe het samenwerkt met HTML. Het voelde als een goede manier om mijn geheugen op te frissen en weer in de flow te komen.
