Praktijkopdracht 6: CSS Bootstrap

Wat We Doen

Voor de opdracht moesten we een website maken met alleen Bootstrap, dus zonder CSS erbij. We kregen screenshots van hoe de site eruit moest zien op verschillende apparaten, zoals een pc, tablet en gsm. Het was de bedoeling om dat zo goed mogelijk na te maken met dingen zoals een navigatiebalk, een hero-sectie, kaarten, een galerij met BaguetteBox en een carrousel. Alles moest werken met de Bootstrap classes, dus we moesten goed weten hoe dat systeem in elkaar zit.

Wat Ik Ervan Vond

Ik had nog nooit zo gewerkt, dus ik moest echt veel opzoeken. Soms gebruikte ik de verkeerde classes, waardoor het er raar uitzag. Maar bugs of errors had ik gelukkig bijna niet, want meestal werkte het meteen als ik de juiste dingen gebruikte. Uiteindelijk heb ik de site helemaal nagemaakt zoals in de opdracht stond. Ik vond het wel chill dat alles uiteindelijk klopte, en ik heb er ook veel uit geleerd. Best trots op hoe het gelukt is!
